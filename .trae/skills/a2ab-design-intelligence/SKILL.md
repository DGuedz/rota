# A2AB Design Intelligence Skill

Esta skill aplica automaticamente as regras rígidas do protocolo VSC para qualquer saída criativa ou visual no repositório.

## Diretrizes Obrigatórias (Non-negotiable)

1. **Aesthetic (Estética)**: "Institutional Brutalism" & "The Sovereign Institution". Mistura da honestidade estrutural do brutalismo com design editorial de alto padrão (high-end).
2. **Colors & UI**: ROTA Bone (warm parchment base), ROTA Black, Signal Gold & Route Amber (highlights). SEM raio de borda (0px border radius). Use elevação tonal (surface-container) ao invés de bordas de 1px ou sombras pesadas.
3. **Typography**: Instrument Serif / Newsreader (Headlines, narrativas), Inter / IBM Plex Sans (UI e Dados, estilo "sala de máquinas").
4. **Themes (Temas)**: Fluxo de valor on-chain, nós de rede (nodes), escrow locking, smart contracts.
5. **Prohibitions (Proibições)**: 
   - ZERO Emojis na saída de texto ou HTML.
   - Sem imagens de "robôs fofos", "cérebros brilhantes" ou azul de internet padrão ("internet blue").
   - Não use termos como "mágica", "chatbot" ou "inteligência artificial" como principal selling point. O selling point é "Liquidação e Trustless Settlement".

## Instruções de Execução

Ao receber uma solicitação do usuário, gere a resposta no formato adequado:

### Se o usuário pedir um **Briefing de Imagem**:
Gere o prompt exato para um gerador de imagens focando na paleta da instituição (ROTA Bone, ROTA Black) e em estruturas técnicas de diagrama.
**Formato**: `[IMAGE BRIEFING] STYLE: Institutional Brutalism, tactile... SUBJECT: ... CONSTRAINTS: NO robots...`

### Se o usuário pedir uma **Estrutura de Landing Page / UI**:
Gere a estrutura HTML/TailwindCSS (ou markdown wireframe) usando as regras acima.
**Estilos**: Use `bg-[#fbf9f2]` (ROTA Bone), `text-[#0a0a0a]` (ROTA Black), botões quadrados (`rounded-none`), fontes Serif para cabeçalhos. Não use divisores horizontais opacos, use espaços em branco ou variação de tons para separar seções.

### Se o usuário pedir um **Script de Vídeo**:
Siga o arco narrativo de conversão técnica.
1. Technical Hook.
2. The Friction.
3. Hybrid Architecture (Solution).
4. On-chain Proof.
5. CTA (Execute x402).
